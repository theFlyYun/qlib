# Model Validation

## 目标

建立模型验证习惯：先看样本外指标，再谈预测名单。

## 验证层级

### 第一层：训练指标

LightGBM 当前使用：

```text
loss = mse
metric = l2
```

它衡量预测值和标签之间的均方误差。

### 第二层：信号指标

重点看：

```text
IC
Rank IC
ICIR
Rank ICIR
```

这些指标判断模型分数是否能解释未来横截面收益。

### 第三层：策略回测

模型分数必须进入策略才能判断是否可交易。

需要看：

```text
年化收益
信息比率
最大回撤
换手率
成本前收益
成本后收益
```

## 当前结论

今天的 Nasdaq Qlib 模型只完成了前两层，尚未完成 TopK 成本后回测。

后续每次实验必须保留 `resolved_config.yaml`。如果没有这份实际配置记录，就无法确认股票池、标签、特征、切分和模型参数，也就不可复盘。
