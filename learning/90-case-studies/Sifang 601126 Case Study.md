# Sifang 601126 Case Study

## 目标

用单只股票案例理解历史走势、收益、波动和回撤。

## 输入数据

股票：

```text
四方股份 601126.SH
```

本地分析报告：

```text
analysis/sifang_601126/601126_sifang_analysis.md
```

## 学到什么

单只股票适合学习：

```text
历史走势
累计收益
波动率
最大回撤
近期涨跌幅
均线和动量
```

但单只股票不适合完整发挥 Qlib 的选股能力，因为 Qlib 更擅长在股票池中做横截面排序。

## 遗留问题

- 单股票分析无法回答“相对其他股票是否更好”。
- 需要同行业或全市场股票池对比。
- 需要加入财报和行业数据。
