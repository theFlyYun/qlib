# Data Scope And Sources

## 目标

为 40 年数据和多源特征扩展建立数据口径。完整升级方案看 [[Data Source Upgrade Plan]]。

## 为什么要做

数据质量决定研究上限。模型再复杂，也救不了口径错误、未来函数、幸存者偏差或错误复权。

## 当前数据源定位

今天的 Nasdaq 实验使用免费公开接口，适合作学习，不适合作正式研究基线。

主要问题：

```text
历史短
复权口径不够清楚
缺少退市股票
缺少动态成分股
缺少 PIT 财报
缺少专业行业分类
缺少可审计的数据版本
```

Norgate 是第一条正在工程化接入的升级路线。它先解决价格、复权、退市股票和历史指数成分问题，详见 [[Norgate Data Integration]]。

当前 Nasdaq 学习流水线已经加入轻量证券主数据层，详见 [[Security Master Data]]。它能显式记录普通股、ADR/ADS、warrant、preferred、debt、unit 等类型，但仍不是历史 PIT 主数据。

## 数据验收标准

每个数据源必须记录：

```text
覆盖市场
覆盖时间
频率
复权方式
是否保留原始价格
是否包含退市股票
是否支持动态成分
是否支持 PIT
缺失值规则
更新时间
授权和使用限制
可复现方式
```

## 学习数据 vs 严谨研究数据

学习数据可以接受不完美，但必须在报告里标注限制。

严谨研究数据必须回答：

```text
当时有哪些股票可以买
当时能看到哪些财报和事件
当时价格是否已经正确处理分红拆股
退市股票是否仍在历史样本里
每次实验能否用同一数据版本重跑
```

## 优先级

```text
第一优先级：价格、复权、退市、动态股票池
第二优先级：行业、估值、PIT 财报
第三优先级：宏观状态变量
第四优先级：新闻和文本情绪
```
