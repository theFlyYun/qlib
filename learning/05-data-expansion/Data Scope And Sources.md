# Data Scope And Sources

## 目标

为 40 年数据和多源特征扩展建立数据验收标准。

## 为什么要做

数据质量决定研究上限。模型再复杂，也救不了口径错误或未来函数。

## 数据验收表

每个数据源必须记录：

```text
覆盖市场
覆盖时间
频率
复权方式
是否包含退市股票
是否支持动态成分
是否支持 PIT
缺失值规则
更新时间
授权和使用限制
可复现方式
```

## 当前数据源问题

今天的 Nasdaq 实验使用免费公开接口，适合作学习，不适合作正式研究基线。

主要问题：

```text
历史短
复权口径不够清楚
缺少退市股票
缺少动态成分股
缺少 PIT 财报
缺少专业行业分类
```
