# Qlib Quant Learning

这是面向量化交易学习的中文路线文档。主线不讲技术实现细节，而是按“市场问题 -> 数据源 -> 信号原理 -> 策略构造 -> 回测评估”的顺序递进。

## iOS 阅读方式

推荐使用 GitHub：

1. 把当前分支 push 到 GitHub。
2. 在 iPhone 或 iPad 打开 GitHub App 或 Safari。
3. 进入 `learning/README.md`。
4. 从学习路线开始阅读。

这种方式不占 iCloud 空间，并且文档和代码版本保持一致。

## 怎么读

先读 Index，再按周推进。每周只要求你理解一个核心问题：

- 第 1 周：量化交易到底在解决什么问题。
- 第 2 周：数据从哪里来，为什么数据质量决定策略上限。
- 第 3 周：信号、因子和模型如何表达市场假设。
- 第 4 周：策略如何落地，回测如何判断真假。

`Qlib Commands` 是附录，只在需要跑实验时看。
