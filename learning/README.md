# Qlib Quant Learning

这是本仓库内的 Qlib 量化交易学习文档入口。相比 iCloud vault，这个版本更适合通过 GitHub 同步和在 iOS 端阅读。

## iOS 阅读方式

推荐使用 GitHub：

1. 把当前分支 push 到 GitHub。
2. 在 iPhone 或 iPad 安装 GitHub App，或直接用 Safari 打开仓库。
3. 进入 `learning/README.md`。
4. 从下面的学习路线开始阅读。

这种方式不占 iCloud 空间，并且学习文档跟代码版本保持一致。
