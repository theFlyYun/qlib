# Qlib Quant Learning

这是面向量化交易学习的中文路线文档。主线不讲技术实现细节，而是按“市场问题 -> 数据源 -> 信号原理 -> 策略构造 -> 回测评估 -> 组合风控 -> 实盘约束”的顺序递进。

## iOS 阅读方式

推荐使用 GitHub：

1. 把当前分支 push 到 GitHub。
2. 在 iPhone 或 iPad 打开 GitHub App 或 Safari。
3. 进入 `learning/README.md`。
4. 从学习路线开始阅读。

这种方式不占 iCloud 空间，并且文档和代码版本保持一致。

## 怎么读

先读 Roadmap，再读 Index，然后按周推进。每周只要求你理解一个核心问题：

- 第 1-2 周：市场和数据。
- 第 3-4 周：收益风险和因子。
- 第 5-6 周：策略和回测。
- 第 7-8 周：组合和风控。
- 第 9-10 周：机器学习。
- 第 11-12 周：实盘约束和复盘。

`Qlib Commands` 是附录，只在需要跑实验时看。

今天的 Nasdaq/Qlib 模型学习复盘请先读 `Learning Session - 2026-05-17 Nasdaq Qlib Model`。
