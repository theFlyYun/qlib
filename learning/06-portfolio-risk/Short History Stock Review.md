# Short History Stock Review

## 这一阶段要回答什么

阶段 5.8C 已经证明：简单给短历史股票扣分，并不会提高收益，严格流动性门槛还会明显损失超额收益。

所以阶段 5.9 不继续调惩罚参数，而是反过来问：

```text
短历史股票到底是不是问题？
如果是问题，是所有短历史股票都有问题，还是少数行业、少数公司、少数状态有问题？
```

本阶段只做解释性复盘，不重新训练模型，不改变选股规则。

## 复盘口径

主基线：

```text
raw_score_sector_cap_2_top10
```

也就是当前表现最好的行业约束版本：

```text
单 sector 最多 2 只
单 industry 最多 2 只
不做短历史 score 校准
```

复盘直接读取该策略的真实回测持仓：

```text
strategy_comparison/raw_score_sector_cap_2_top10/backtest_positions.csv
```

这很重要：5.9 解释的是已经发生的持仓结果，不重新计算收益，也不重新生成一套选股。

## 复盘对象

目标短历史桶：

```text
lt_2y：180 到 503 个交易日
2_5y：504 到 1259 个交易日
```

参照桶：

```text
full_10y：至少 2400 个交易日
5_10y：1260 到 2399 个交易日
```

每个桶内再按实际持仓收益分出：

```text
bucket_winners：该桶内未来收益最高的一组
bucket_losers：该桶内未来收益最低的一组
```

这里的收益来自回测口径：信号日后 1 个交易日入场，持有 5 个交易日。

```text
短历史输家的共同问题更偏向高估值、亏损公司和局部行业，而不是单纯历史短。
```

## 行业拆分

短历史净贡献最弱的 sector：

```text
2_5y / Finance：
持仓 12 次，平均收益 -5.97%，胜率 16.67%，净贡献 -7.27%。

lt_2y / Technology：
持仓 14 次，平均收益 0.14%，胜率 35.71%，净贡献 0.08%。

lt_2y / Consumer Discretionary：
持仓 46 次，平均收益 0.10%，胜率 56.52%，净贡献 0.11%。
```

这说明真正明显的负贡献集中在 `2_5y / Finance`，而不是所有 `lt_2y` 或所有短历史股票。

短历史净贡献较强的 sector：

```text
2_5y / Basic Materials：
持仓 5 次，平均收益 13.29%，胜率 100%，净贡献 6.61%。

2_5y / Industrials：
持仓 7 次，平均收益 6.51%，胜率 57.14%，净贡献 4.51%。

lt_2y / Industrials：
持仓 26 次，平均收益 1.71%，胜率 42.31%，净贡献 4.26%。
```

样本数不大，但方向很明确：短历史机会和短历史风险都有明显行业差异。

## 赢家和输家的特征差异

差异最大的几个方向：

```text
lt_2y 赢家的市值明显高于 lt_2y 输家。
lt_2y 赢家的最新价格明显高于 lt_2y 输家。
lt_2y 赢家的 20 日成交额也高于 lt_2y 输家。
2_5y 输家中亏损公司占比非常高。
```

从风险标签看：

```text
lt_2y 输家：
低流动性占比 33.33%
高估值占比 38.10%
亏损公司占比 42.86%

2_5y 输家：
低流动性占比 8.51%
高估值占比 42.55%
亏损公司占比 82.98%
```

这说明 `2_5y` 的问题更像是“亏损公司 + 高估值”的基本面风险，而不是简单流动性风险。

## 当前判断

不建议剔除短历史股票。

更合理的判断是：

```text
继续保留短历史桶名额。
不要继续加大统一短历史惩罚。
下一步应该做 sector-specific 短历史规则。
```

原因：

```text
lt_2y 和 2_5y 都是正净贡献。
短历史输家并不平均分布，而是集中在特定行业和基本面状态。
统一扣分会误伤 Basic Materials、Industrials 等短历史贡献来源。
```

## 下一步建议

下一阶段可以做阶段 5.10：短历史行业约束。

第一版建议不是复杂模型，而是规则对照：

```text
保持短历史总名额不变
限制 2_5y / Finance 最多 0 或 1 只
限制短历史亏损公司进入 Top10 的数量
观察是否降低最差单票亏损和最大回撤
```

如果这个对照有效，再考虑：

```text
sector-specific 短历史名额
sector-specific 短历史惩罚
短历史股票单独模型
```
